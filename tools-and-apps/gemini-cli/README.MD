# Gemini CLI


## Installation

### Windows

Open PowerShell.

Install nodejs:
```winget install -e --id OpenJS.NodeJS```<br>

Reopen PowerShell.

Install gemini-cli:
```npm install -g @google/gemini-cli```<br>
